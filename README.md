# ProGamez
GameZone Sales Analysis 2019-2021. 
An analysis for a consumer electronics multinational with multiple product offerings. This analysis tracks sales performance using order-level data, including timestamps, product attributes, marketing channels, and regional information.
With an average order value (AOV) of $281, the company attracts high-intent buyers, making it essential to understand purchase patterns, product attachment behavior, and opportunities to increase lifetime value.
The analysis focuses on uncovering trends, customer insights, and business recommendations that support growth in revenue, margin, and operational efficiency.
